source

load_market_data_file

 load_market_data_file (market_data_file:pathlib.Path)

Loads a single market data file

Type Details
market_data_file Path csv file to laod data from
Returns DataFrame data frame with transactions

source

get_symbols

 get_symbols (market_data_path:pathlib.Path)

Returns a list of all symbols available at given market data dump directory

Type Details
market_data_path Path directory with market data
Returns typing.List[str] List of symbols that have market data available

source

load_all_market_data_files_for_symbol

 load_all_market_data_files_for_symbol (market_data_path:pathlib.Path,
                                        symbol:str)

Loads all data for the given symbol

Type Details
market_data_path Path directory with market data
symbol str name of the symbol
Returns DataFrame dataframe with transactions

make_sequential_stream

 make_sequential_stream (data_frames:List[pandas.core.frame.DataFrame])

Takes a dictionary of data frames and merges them together according to the timestamps (index).

Type Details
data_frames typing.List[pandas.core.frame.DataFrame] frames with transacations
Returns DataFrame sequential streamX
