Exploring distributed computation using dask platform.

Dask (https://www.dask.org) seems to be an easy approach how to scale up the calculation that are done using pytorch and numpy.

Dask has a similar interface as numpy and torch. Dask breaks up large data frames into sections and calculations are automatically distributed across many procesess.
